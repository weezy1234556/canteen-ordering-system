# Canteen Ordering System

A web application that allows students to order food from a school canteen
and gives administrators a dashboard to manage orders and menu items.

🔗 **Live Demo:** https://weezy1234556.github.io/canteen-ordering-system

## Features
- Students can browse menu and add items to cart
- Cart management with order summary
- Admin dashboard to manage orders
- Admin panel to add new menu items
- Clean and easy to use interface

## Built With
- HTML5
- CSS3
- JavaScript

## Pages
| Page | Description |
|------|-------------|
| index.html | Homepage / menu for students |
| cart.html | Student cart and order summary |
| admin.html | Admin login |
| admin-orders.html | Admin orders management |
| add-items.html | Admin add menu items |
